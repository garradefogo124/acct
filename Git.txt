==> COMANDOS <==
- git init --> inicia o git
- git add --> adiciona arquivos
- git commit -m "message" --> faz o commit com a mensagem e deixa pronto para enviar
- git push -u origin [name] --> envia para o repositório desejado
- git push --> envia nas vezes seguintes sem precisar do comando anterior
- git checkout -b [name] --> troca o lugar do commit para uma nova branch que vai ser criada
- git checkout [name] --> troca o lugar do commit para uma branch que já existe
- git clone [url] --> clona repositório online
  
==> PASSO A PASSO PARA FAZER UMA ENTREGA <==
1. Abrir Git Bash na pasta desejada;
2. git clone [url-do-repositorio-base];
3. Entrar na pasta criada e abrir o git bash de novo;
4. git checkout -b [nome-da-nova-branch];
5. git add .;
6. git commit -m "mensagem sobre o que foi feito";
7. git add -u origin [nome-da-branch-que-foi-criada];
